程序说明：本程序是自定义大数据表格，传入数据后，自行创建表格。
辅助说明：
	（默认） 1、表头、表身、表尾是分开定义；（√）
	（默认） 2、出现因无法一次性展示，出现横向滚动条，滚动横向滚动条，表头和表身需联动；（√）
	
	（默认） 3、列鼠标悬浮时显示全部内容；（√）
	（用户） 4、列宽、列头、列数、列转换方式由用户控制；（√）
	（默认） 5、列是否存在可复制的情况由用户控制；（√）
	（用户） 6、列是否允许在使用时自由调整宽度由用户控制；（×）
	
	（默认） 7、奇偶行颜色变化；（√）
	（用户） 8、鼠标悬浮行时有颜色变化由用户控制；（√）

	（默认） 9、支持滚轮、上下键滚动表格；（√）

文件说明：
	CSS ：样式
		BaseStyle.css	 ：表格的基础样式
		UserSetStyle.css ：用户使用时自行修改的样式
	JS  ：js文件
		BaseFunction.js ：表格的基础内容创建
		Loading.js		：表格加载时函数，展示未用到
		UserSet.js		：表格使用时需用数据及调用函数
	index.html ：界面内容

使用说明：
	1、创建div，然后根据userSet引入数据
	2、引用index.html说明的引用文件
	3、就能展示出表格及内容
其他说明：
	无

程序说明：本程序是自定义表格，传入数据后，自行创建表格。
辅助说明：
	（默认） 1、表头、表身、表尾是分开定义；（√）
	（默认） 2、出现因无法一次性展示，出现横向滚动条，滚动横向滚动条，表头和表身需联动；（√）
	
	（默认） 3、列鼠标悬浮时显示全部内容；（√）
	（用户） 4、列宽、列头、列数、列转换方式由用户控制；（√）
	（默认） 5、列是否存在可复制的情况；（√）
	（用户） 6、列是否允许在使用时自由调整宽度由用户控制；（×）
	
	（默认） 8、奇偶行颜色变化；（√）
	（用户） 9、鼠标悬浮行时有颜色变化由用户控制；（√）
	（用户）10、列的第一项默认为勾选框区域，但隐藏或显示、单选还是多选由用户控制；（√）
	（用户）11、鼠标悬浮行，右键允许选择删除、隐藏，此功能由用户控制显示内容；（√）

	（用户）12、是否存在分页情况由用户控制；（√）

文件说明：
	CSS ：样式
		BaseStyle.css	    ：表格的基础样式
		UserSetStyle.css	：用户使用时自行修改的样式
	JS  ：js文件
		BaseFunctionList.js	：表格的基础内容创建
		UserSet.js			：表格使用时需用数据及调用函数
	View：视图
		index.html ：界面内容

使用说明：
	1、总数据量请在1万之内，否则分页会出现问题。请自行修改
其他说明：
	此程序仅能用于基础的展示表格等作用，部分功能待维护；

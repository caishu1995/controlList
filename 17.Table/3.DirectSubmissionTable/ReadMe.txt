程序说明：本程序是自定义表格，传入数据后，自行创建表格。表格适合操作后及时提交的情况。
功能说明： 
	1、表格尾部有新增按钮，点击新增则新增一行，显示均为输入框;
	2、行内有修改按钮，点击按钮后整行均变成输入框的形式，再点击行内的保存按钮，则保存修改;
	3、行内有删除按钮，点击按钮后整行删除;
辅助说明：
	（默认） 1、表头、表身是分开定义；（√）
	（默认） 2、出现因无法一次性展示，出现横向滚动条，滚动横向滚动条，表头和表身需联动；（√）
	———————————————————————————————————————————————————
	（默认） 3、列鼠标悬浮时显示全部内容；（√）
	（默认） 4、列内容可复制；（√）
	（用户） 5、列宽、列头、列数、列转换方式由用户控制；（√）
	———————————————————————————————————————————————————
	（默认） 6、奇偶行颜色变化；（√）
	（默认） 7、鼠标悬浮行，有颜色变化；（√）
	（默认） 8、鼠标悬浮行，右键允许选择删除、修改、保存；（√）
	（用户） 9、行内存在修改、删除按钮，当修改时更改为保存、删除按钮；（√）
	（用户）10、存在新增按钮行，点击新增按钮，新增一行内容（√）
	———————————————————————————————————————————————————
	（用户）11、允许用户添加验证函数；（√）
	（用户）12、用户控制行id的来源；（√）
	———————————————————————————————————————————————————
	（默认）13、控制多个时不会相互影响；（√）

文件说明：
	Style ：样式
		BaseStyle.css	：表格的基础样式，此内容不需要修改
		UserSetStyle.css	：用户使用时自行修改的样式

	JS  ：js文件
		BaseJS.js	：表格的类，此内容不需要修改
		UserSet.js	：表格使用时需用数据及调用函数，用户可以执行修改

    index.html ：界面内容

使用说明：
	使用时请给一个div,并必须给一个唯一标识此表格的id，同时选择大版还是小版。
	请按UserSet.js所示创建对象并调用函数
	修改UserSetStyle，为每列设置宽度，并计算总宽度
	就能展示出表格及内容

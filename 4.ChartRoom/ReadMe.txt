项目名称： 聊天室
知识范围： js css html
版本号  ： 1.1

程序说明：聊天室界面
功能说明：
    1、登入 登出
    2、联系人列表
    3、展示信息
    4、发送信息
辅助说明：
    （默认） 1、未登录时，鼠标移入，显示输入框、登入按钮（√）
    （默认） 2、未登录时，输入名称，点击登入按钮，完成登录（√）
    （默认） 3、登录后，鼠标移入，显示名称、登出按钮（√）
    （默认） 4、登录后，更新联系人列表、成员列表、聊天区（√）
    （默认） 5、登出后，清空联系人列表、成员列表、聊天区（×）

    （默认） 6、左侧展示联系人列表、成员列表（√）
    （默认） 7、联系人列表点击后更新成员列表、聊天区（√）

    （默认） 6、右侧为讨论组具体信息（√）
    （默认） 7、右侧支持状态展示、具体信息查看（√）
    （默认） 8、右侧支持发送信息（√）
    （默认） 9、修改默认滚动条样式，鼠标放入内容区，显示滚动条；鼠标放入滚动条区，改变背景（×）

文件说明：
    chartRoom.js : 聊天室js
    chartRoom.css: 聊天室css
    index.html   : 主界面

使用说明：打开直接使用
引用说明：把全部内容引用
其他说明：无

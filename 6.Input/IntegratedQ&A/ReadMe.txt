项目名称： 快速问答输入框
知识范围： html css js jquery
版本号  ： 1.1

程序说明：用统一的输入框，完成多个问题的快速回答。
功能说明：
    1、根据js数据创建页面
    2、每个问题回答后有验证
    3、问题回答正确，进度条增加，进度数字也增加
辅助说明：
    （默认） 1、根据js创建页面,包括问题 答案填写区 进度条 进度数字 验证提示（√）
    （默认） 2、一次只展示一个问题（√）
    （默认） 3、输入框自动获得焦点（√）
    （默认） 4、点击下一题，更改数据 问题 进度条 进度数字（√）
    （默认） 5、点击下一题，自动进行验证（√）
    （默认） 6、js存储数据，并在问答完成后，将结果传出（√）
    （默认） 1、根据js创建页面（√）
    （默认） 1、根据js创建页面（√）
    （默认） 1、根据js创建页面（√）
    （用户） 2、用户自选是否填写验证函数（√）

文件说明：
    index.html 主界面
    next.png   下一题的图标
    QA.css     问答样式
    QA.js      问答的js代码

使用说明： 直接打开即可
引用说明： 导入png，引用js css，在界面中类似index,html创建div。并使用js创建对象，即可使用
其他说明： 无

注意：     无

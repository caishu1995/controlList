项目名称： hot样式
知识范围： jQuery js css html
版本号  ： 1.2

程序说明：允许下拉选择，且修改 5块*(2或3位) 内容
功能说明：
    1、上下左右回车删除
    2、数字
辅助说明：
    -- 块
    （默认） 1、点击块，选中当前框。（√）
    （默认） 2、点击块，显示下拉列表。（√）
    （默认） 3、左右键，类似上一个或下一个单击(1)。（√）
    （默认） 4、0-9支持两种按键，最多3位，超过则替换第一位。（√）
    （默认） 5、数字个数超过三个，触发7。（√）
    （默认） 6、回车，触发7。（√）
    （默认） 7、选中，获得内容，并清空后面的内容。（√）
    （默认） 8、如果选中的是最后一位，回车或超过三位数，则类似其他位置的点击(15)。（√）
    （默认） 9、删除，删除最后一位内容。（√）

    -- 下拉列表
    （默认） 10、点击，给当前项增加选中样式、去除其他项的样式。（√）
    （默认） 11、双击，类似块的回车(6)。（√）
    （默认） 12、上下键，类似上一个或下一个单击(10)。（√）
    （默认） 13、下键，如果之前没有选中列表中内容，则选中下拉列表第一个。（√）
    （默认） 14、回车，删除类，隐藏下拉列表，其他类似块的回车，(由于绑定在document上，表示也会触发块的回车)。（√）

    -- 页面内按键
    （默认） 15、点击，如果不在指定区域，则恢复成未选择的状态。（√）


文件说明：
    JS  ：js文件
        jquery-1.10.2.js  ：jQuery的文件
        specialInput.js   ：需要引入的文件
    CSS : css文件
        specialInput.css  ：需要引入的样式
    index.html：主界面

使用说明： 打开即可看
引用说明： 引用外部文件，然后初始化

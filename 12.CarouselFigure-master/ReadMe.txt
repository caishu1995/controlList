项目名称： 幻灯片轮转
知识范围： HTML css js jQuery
版本号  ： 1.2

程序说明：图片轮播，左侧是图片组，右侧为大图，并可进入不同界面
功能说明：
    1、图片轮播，左侧是图片组，右侧为大图，并可进入不同界面
辅助说明：
    （默认） 1、左侧图片轮播，改变边框颜色，改变滚动条位置 （√）
    （默认） 2、左侧，鼠标悬浮，放大。双击，显示该图 （√）
    （默认） 3、右侧显示图片 （√）
    （默认） 4、右侧，悬浮显示界面名称、进入按钮 （√）
    （默认） 5、右侧，点击进入按钮，打开对应界面 （√）

文件说明：
	img : 图片集合
    JS  ：js文件
        CarouselFigure.js ：轮播的js
        index.js          ：根据权限返回的数据创建左侧轮播列表的内容
        jquery-1.10.2.min.js：jQuery的文件

    index.html：主界面
    index.css :样式文件

使用说明： 打开即可预览
引用说明： 引用js，按html结构创建页面，并初始化
其他说明： 一个Html中，仅支持有一个leftDiv rightDiv

问题： 无
注意： 无

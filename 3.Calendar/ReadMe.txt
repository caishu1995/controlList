项目名称： 日历功能
知识范围： js css html
版本号  ： 1.1

程序说明：日历功能
功能说明：
    1、点击显示当前日期
    2、显示当天日期
    3、年份 月份 选择
辅助说明：
    （默认） 1、根据百度默认日历，做出类似的样式 （√）
    （默认） 2、周六、周天、当天、选中天特殊标记 （√）
    （默认） 3、悬浮点击样式 （√）
    （默认） 4、根据本月份数据显示不同内容 （√）
    （默认） 5、点击日期，右侧显示详细内容 （√）
    （默认） 6、选择年份，上一年 下一年 选择年 （√）
    （默认） 7、选择月份，上一月 下一月 选择月 （√）

文件说明：
    CalendarStyle.css ：样式文件
    CalendarClass.js  ：日历类
    calendarShow.html : 界面

使用说明：
    1、定义一个div
    2、定义个对象，提供一个获得日期的方法，调用初始化

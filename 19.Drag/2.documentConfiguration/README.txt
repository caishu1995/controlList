项目名称： 类似文档的拖拽书写内容
知识范围： html css js
版本号  ： 1.1

程序说明：类似文档的，可以拖拽区域 写字区、表格等，实现模板的结构
功能说明：
    1、可以拖拽文字，支持相关内容的更改。
    2、可以拖拽表格，支持相关内容的更改。
辅助说明：
    （默认） 1、支持拖拽文字（√）
    （默认） 2、支持文字字体字号内容修改（√）
    （默认） 3、支持拖拽表格（√）
    （默认） 4、支持表格列名列宽修改（√）

    （默认） 5、支持初始化（√）
    （默认） 6、支持保存，保存后获得所有信息（√）

文件说明：
	documentConfig.css：样式文件
	documentConfig.js : js文件
	index.html        : 主页
使用说明：直接打开，并拖动工具箱内容到指定区域。单击即可修改内容

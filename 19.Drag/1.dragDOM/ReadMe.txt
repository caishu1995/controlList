    （默认） 11、支持IE FF google（√）

问题：
    1、画出选择的框
    答：onmousedown onmouseup onmousemove，按下开始画框，移动，改变框大小，松开，选中

    2、右键如何禁用默认的内容
    答：document.oncontextmenu = function(ev) { ev.preventDefault(); };

    3、判断左右键
    答：e.button == 2右键 e.button == 1滑轮 e.button == 0左键

    4、dataset IE中报错
    答：ie中dataset不存在，不能直接使用。所以第一次给dataset赋值的时候都需要检查一下dataset是否存在。

    5、e.dataTransfer.setData("aa","") IE中报错
    答：ie中对于 e.dataTransfer.setData() 第一个参数只认识Text URL，其他的不认识就会报错。但其他浏览器支持

    6、当鼠标进入拖拽内容的div中，框的大小计算会出错
    答：使用绝对定位，不使用相对定位。当进入内部元素的时候，相对定位就不正确。

    7、多选的时候，存在点击清空之前选中的情况
    答：这里存在一个误操作，如果多选的过程中，忘记松开鼠标，则视为拖动。这违背用户本意，所以需要确保拖动时ctrl键没有被按住。
